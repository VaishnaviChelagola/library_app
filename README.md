
## Project Overview
This project is a Library Management Web Application built using Flask where users interact
through HTML pages  for data storage.
It supports following operations:

* User authentication using JWT
* Role-based authorization (Admin and Member)
* Secure access control at page level and action level
* Persistent storage using SQLite3


---

## Setup Instructions

### 1. Clone the repository

git clone <https://github.com/VaishnaviChelagola/library_app>


### 2. Install dependencies

pip install -r requirements.txt


### 3. Run the Application

From the main directory library-web-app (where `app.py` file is) run:

python app.py

* Website URL: `http://127.0.0.1:5000`
